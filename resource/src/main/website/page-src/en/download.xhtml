System Requirements

  • You need Java 1.5 or higher to run betterFORM

Run betterFORM

  1. Switch to your installation directory
  2. Execute the betterform.jar by double-clicking it. A small application window with the betterFORM logo will appear
  3. Click on the icon to start your browser with the betterFORM start page

Getting started

To get started we recommend to have a look at the XFormsFeatureExplorer which shows live examples of all available controls and containers along with markup and descriptions. The FeatureExplorer will constantly be extended and updated as betterFORM moves on.

The User Guide is a must-read for every new user and especially focuses on XForms authoring, styling and page layout. It should give an overview of what the platform offers out of the box.

The Developer Guide is for architects and developers seeking information about the architecture of betterFORM, building from sourcecode, the various ways to extend the functionality and usage of the Java API.

Get the sources

The source code of betterFORM is available from Github.
