Das betterFORM Projekt

Wir sind ein Team von Softwareentwicklern, das täglich an der Verbesserung der betterFORM Produkte arbeitet und sie gleichzeitig in verschiedensten Projekten professionell einsetzt. Unsere Philosophie ist Offenheit: Open-Source, öffentliche, Hersteller-unabhängige Standards sowie aufgeschlossene Kommunikation mit unseren Kunden und Anwendern führen zu langlebigen, erweiterbaren Lösungen.

Das betterFORM Team:

Joern Turner ist ein XML Applikation Architekt mit spezieller Expertise im W3C XForms Standard. Im Jahr 2001 gründete er das Chiba Projekt. Dies war die erste serverseitige XForms Implementierung, die in jedem Standard-Browser ohne zusätzliche Software lief.

Während der letzten 20 Jahre arbeitete er als Berater, Software Architekt, Projektleiter und Entwickler und realisierte Enterprise Anwendungen mit Java und XML-Technologien für eine breite Palette von Anwendungsbereichen.

Lars Windauer ist ein XML Experte mit dem Hauptaugenmerk auf der Entwicklung von XForms Anwendungen. Lars trat 2006 dem Chiba Projekt bei und gründete Anfang 2009 zusammen mit Joern Turner das betterFORM Project.

Lars entwirft und setzt XForms basierte Anwendungen um. Hierbei kommen auch weitere XML Kerntechnologien wie z.B. XSLT, XSD aber auch Java, JavaScript und CSS zum Einsatz. Er betreut Kunden durch den gesamten Software-Lebenszyklus. Angefangen beim Aufnehmen der Anforderungen bis zur Auslieferung der fertigen Software.

Tobi Krebs ist ein Software Architekt und XForms Experte, der mit Hilfe von XML-Technologien (wie XForms, XSLT, XQuery und XML-basierten Datenbanken), Java and JavaScript weltweit Software-Lösungen für Kunden entwirft und implementiert. Mitte 2009 trat er dem betterFORM Project bei und ist seitdem insbesondere in der Integration von XML Datenbanken tätig.

Zuvor arbeitete er als Java Entwickler für die "Gesellschaft für Informationssysteme und Prozessautomation mbH".

Fabian Otto ist seit seiner Kindheit Computernutzer und Programmierer. Er hat eine Vorliebe für merkwürdige und dynamische Programiersprachen und nutzt ausschließlich *NIX - ähnliche Betriebssysteme. Er arbeitet seit Herbst 2009 im betterFORM Team.
