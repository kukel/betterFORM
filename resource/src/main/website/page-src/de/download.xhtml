Systemanforderungen

  • Es wird Java 1.6 oder höher benötigt, um betterFORM auszuführen.

betterFORM installieren

  1. Laden Sie betterFORM durch Anklicken des Links 'download now'
  2. Doppelklicken Sie die heruntergeladene Datei, um den Installer zu starten
  3. Folgen Sie den Anweisungen des Installers

betterFORM starten

  1. Wechseln Sie in das Installations-Verzeichnis
  2. Mit einem Doppelklick auf 'betterform.jar' führen Sie betterFORM aus. Es erscheint ein kleines Anwendungsfenster mit dem betterFORM Logo.
  3. Klicken Sie auf das Logo, um Ihren Browser mit der betterFORM Startseite zu öffnen.
Diese Anweisungen gelten für den Standalone Server sowie die betterFORM XML Suite. Bei Verwendung als Webapplication beachten Sie die Instruktionen für Ihren jeweiligen Server.

Erste Schritte

Um sich mit betterFORM vertraut zu machen, empfehlen wir den XFormsFeatureExplorer. Dieser zeigt Live-Beispiele aller Controls und Container mit Markup-Beispielen und Beschreibungen. Der Feature Explorer wird ständig um neue Beispiele erweitert, um neue betterFORM Funktionen zu demonstrieren.

Der User Guide sollte von jedem Anwender gelesen werden und wendet sich insbesondere an XForms Autoren, die die Grundlagen von betterFORM kennenlernen möchten.

Der Developer Guide richtet sich an Architekten und Enwickler, die Informationen über die Architektur und Implementierungsdetails von betterFORM suchen. Hier wird beschrieben, wie betterFORM compiliert wird, wie und wo Erweiterungen gemacht werden können und wie die API genutzt werden kann, um komplexe Anforderungen zu bedienen.
