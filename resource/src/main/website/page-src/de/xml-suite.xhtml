XML from front to back

Die betterFORM XML Suite vereinigt die Vorteile von XML, XForms, XQuery, XSLT, XPath, XML Schema, XHTML und anderen in einem einzigen, freien Produkt.

Von der Benutzeroberfläche bis zur Datenspeicherung wird durchgängig XML verwendet. So werden teure Datenkonvertierungen vermieden und ein einzigartiges, flüssiges Arbeiten ermöglicht.

Erstellen Sie Rich Internet Applications

Die betterFORM XML Suite enthält alles, was eine Rich Internet Application benötigt. Konzentrieren Sie sich auf Ihre Anforderungen und profitieren Sie vom eingebauten XForms Toolkit und der exzellenten eXist XML Datenbank.
