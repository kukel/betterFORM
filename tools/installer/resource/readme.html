Installer Readme



Quick install

For a quick install choose the components betterFORM XRX and follow the install instructions. After the installation finishes step into the install directory and execute betterFORM.jar



betterFORM WAR

The betterFORM project. If no other component is selected, betterFORM is installed as web application archive (WAR file) at the choose directory.



betterFORM standalone

The betterFORM standalone component installs an embedded Jetty server betterFORM runs in. After installation switch to the chosen install directory and execute betterFORM.jar to start the server



betterFORM XRX

The betterFORM XRX component installs betterForm and eXist as a ready to go XRX platform. After installation switch to the chosen install directory and execute betterFORM.jar or the startup.sh/startup.bat from the bin directory

Developer Pack

adds all JavaScript sources uncompiled for full debug capabilities. betterFORM recommends Firebug as client side debugging tool



Forms

This installs the betterForm's demo, reference and test forms. For the XRX platform these forms are already installed into the db



Testsuite1.1

The W3C XForms Conformance Test Suite.
This feature is currently not available for the XRX platform.



* assuming you run betterFORM within the embedded Jetty Server. If you deploy betterFORM into your own Servlet Container you have to adjust the URLs to the server and port betterFORM runs in
