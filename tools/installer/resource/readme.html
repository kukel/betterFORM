Installer Readme



Quick install

For a quick install choose the components betterFORM XML Suite and follow the install instructions.


After the installation finishes step into the install directory and execute betterFORM.jar or the startup.sh/startup.bat from the bin directory



betterFORM XML Suite

The betterFORM XML Suite component installs the betterFORM XForms Toolkit and eXist XML database as a ready to go XRX platform.


After installation switch to the chosen install directory and execute betterFORM.jar or the startup.sh/startup.bat from the bin directory

betterFORM XForms Toolkit

betterFORM XForms is a XForms engine for advanced, dynamic forms that run in every major browser without any plugins. You can choose between a standalone version (bundled with jetty) or deploy it to a web container of your choice.


The "standalone" version provides it own uniqe starter: after the installation switch to the chosen install directory and execute betterFORM.jar to start it


Developer Pack

adds all JavaScript sources uncompiled for full debug capabilities. betterFORM recommends Firebug as client side debugging tool



Forms

This installs the betterFORM demo, reference and test forms. For the betterFORM XML Suite these forms are already installed.
